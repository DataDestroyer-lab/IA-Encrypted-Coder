# SecureVault

## Status
⚠️ **Still in Development** - This project is a work in progress.

## About
This is just a project I built because I was bored. It's a simple web application with basic authentication and encryption features.

## Login
You can log in by typing **anything** in both the username and password fields. Currently, all logins are authenticated as a regular user.

## Project Structure
- `index.html` - Main HTML file
- `styles.css` - Styling
- `js/` - JavaScript files
  - `app.js` - Main application logic
  - `Encryptdata.js` - Data encryption functionality
